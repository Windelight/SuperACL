## 简介
 本规则有以下两种模式
 * 大陆出国模式(仅代理中国大陆有访问限制的网站)
 * 海外回国模式(代理范围为中国大陆地区的网站)  

本规则可以帮助您  
- [x] 代理在中国大陆被封锁的网站 (eg. Google、 Facebook、 Twitter)
- [x] 代理在中国大陆访问速度较慢或不稳定的网站 (eg. GitHub、 Amazon AWS)
- [x] 代理在中国大陆有访问限制的网站 (eg. Steam商店锁区、 Xbox Live锁区)
